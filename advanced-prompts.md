# Advanced AI Recruitment Prompts

## Deep Candidate Evaluation

```
Analyze this candidate profile deeply. Assess technical capability, communication, leadership potential, and long-term growth. Provide a hiring recommendation with reasoning.
```

---

## Passive Candidate Conversion

```
Write a message to convince a passive candidate to consider a new opportunity. Focus on career growth, impact, and unique value proposition.
```

---

## Candidate Comparison

```
Compare these 2 candidates for the role of [Role]. Provide a side-by-side analysis and recommend the stronger candidate.
```

---

## Hiring Strategy Prompt

```
Act as a Head of Talent. Design a hiring strategy for a startup hiring for [Role]. Include sourcing channels, evaluation steps, and timeline.
```
