Most teams struggle to use AI effectively in hiring.

---

## Prompt Library

### Candidate Sourcing

```
Find 20 candidates for [Role] with experience in [Skill], working in [Industry], based in [Location]. Prioritize candidates from companies similar to [Target Companies].
```

---

### Cold Outreach

```
Write a personalized cold outreach message for a [Job Title] candidate currently working at [Company], highlighting growth opportunity, impact, and why they should consider the role.
```

---

### Resume Screening

```
Evaluate this resume against the job description. Score the candidate from 1–10 based on relevance, experience, and role fit. Highlight strengths and gaps.
```

---

### Interview Questions

```
Generate 10 high-quality interview questions for a [Role]. Include technical, behavioral, and situational questions.
```

---

### Job Description Generator

```
Write a compelling job description for a [Role] at a fast-growing startup. Focus on impact, ownership, and growth opportunities.
```
